Jornada ao Tesouro Perdido na Amazônia

Passo 1: Início da Jornada

Você está em Manaus, a porta de entrada para a Amazônia. Você encontrou um antigo mapa com uma pista.

Passo 2: Escolha da Rota

O mapa indica dois caminhos: um através da selva densa e outro pelo rio.

Passo 3: Desafios da Selva

Você entrou na selva e encontrou um grupo de indígenas amigáveis. Eles oferecem duas opções: uma trilha segura ou um caminho mais curto mas perigoso.

Passo 3: Navegação no Rio

Você optou pelo rio e encontrou um barco. Durante a navegação, você se depara com duas opções: um atalho através de um afluente ou seguir o curso principal.

Passo 4: Trilha Segura

A trilha segura leva você a uma clareira onde você encontra um antigo templo. O templo tem dois caminhos: um para dentro e outro para um esconderijo.

Passo 4: Caminho Perigoso

O caminho perigoso leva você a uma área com animais selvagens. Você encontra uma ponte quebrada com duas opções: consertar a ponte ou procurar uma rota alternativa.

Passo 4: Afluente

O afluente leva você a uma cachoeira. Há um caminho atrás da cachoeira e uma gruta ao lado.

Passo 4: Curso Principal

Seguindo o curso principal, você encontra uma pequena vila. Os habitantes oferecem duas opções: uma pista antiga ou uma nova pista.

Passo 5: Templo

Dentro do templo, você encontra uma sala com três portas: uma dourada, uma de prata e uma de bronze.

Passo 5: Esconderijo

No esconderijo, você encontra uma caixa trancada com um enigma. Resolver o enigma revela uma pista para o próximo passo.

Passo 5: Conserto da Ponte

Consertando a ponte, você consegue atravessar e encontra uma caverna com uma pista que leva a um vale.

Passo 5: Rota Alternativa

Você encontra uma trilha oculta que leva a uma aldeia abandonada com uma pista para o tesouro.

Passo 5: Atrás da Cachoeira

Você encontra uma passagem secreta atrás da cachoeira que leva a um antigo salão.

Passo 5: Gruta

A gruta tem uma entrada secreta com uma armadilha. Você precisa desativar a armadilha para avançar.

Passo 5: Pista Antiga

A pista antiga leva a uma antiga fortaleza. Dentro, você encontra um mapa com uma nova localização para o tesouro.

Passo 5: Nova Pista

A nova pista leva a um labirinto. Você precisa encontrar a saída para avançar.

Passo 6: Porta Dourada

A porta dourada leva a uma sala cheia de tesouros, mas o verdadeiro tesouro está escondido atrás de uma parede falsa.

Passo 6: Porta de Prata

A porta de prata leva a uma sala com um enigma que, se resolvido, revela o caminho para o tesouro.

Passo 6: Porta de Bronze

A porta de bronze leva a uma sala com um mapa que indica a localização do tesouro.

Passo 6: Resolver o Enigma

O enigma revela uma pista final para o tesouro.

Passo 6: Vale

No vale, você encontra uma caverna com um enigma que revela a localização do tesouro.

Passo 6: Aldeia Abandonada

A aldeia abandonada tem um baú com um mapa final para o tesouro.

Passo 6: Salão

No salão, você encontra um velho diário com a localização exata do tesouro.

Passo 6: Desativar Armadilha

Desativando a armadilha, você encontra uma passagem secreta para o tesouro.

Passo 6: Novo Mapa

Seguindo o novo mapa, você encontra a entrada para uma antiga fortaleza onde o tesouro está escondido.

Passo 6: Labirinto

Você encontra a saída do labirinto e a localização final do tesouro.

Passo 7: Tesouro Encontrado!

Parabéns! Você encontrou o tesouro perdido na Amazônia! A aventura foi um sucesso.
